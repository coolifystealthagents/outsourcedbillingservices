# Design summary

- Design concept: Billing claims board
- Layout personality: medical-adjacent board
- Typography: Sora headings with Inter body text
- Color palette: primary #075985, secondary #7dd3fc, accent #f97316, background #f0f9ff, text #082f49, muted #64748b
- Shape language: Slight rounding
- Visual motif: claim cards and audit trails
- Unique layout choices: first sections use Revenue cycle board, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
